# 贡献指南

感谢你对 forum-clipper 项目的关注！欢迎任何形式的贡献。

## 如何开始

1. Fork 本仓库
2. 创建你的功能分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m 'Add some feature'`
4. 推送到分支：`git push origin feature/your-feature`
5. 提交 Pull Request

## 分支策略

- `master` - 开发/测试分支，新功能先推到这里验证
- `main` - 稳定分支，测试通过后才合并

## 提交规范

- 使用清晰的提交信息
- 一次提交只做一件事
- 修复 bug 时关联对应的 issue

## 代码风格

- 使用原生 JavaScript，无框架依赖
- 保持代码简洁，添加必要注释
- 遵循现有的代码结构

## Issue 模板

提交 issue 时请包含：
- 问题描述
- 复现步骤
- 预期行为
- 实际行为
- 环境信息（浏览器版本、操作系统等）
